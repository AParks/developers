Plugin API

Plugins allow you to extend your WizeHive experience by adding new screens or functionality to existing screens within WizeHive. Using our developer tool, create custom dashboards, Gantt charts, reports and more. Plugins include buttons that can send data from an existing form to Zapier, UPS tracking status integration, a task hotlist that follows you from screen to screen and whatever else you think of next.

Why WizeHive?

WizeHive has helped over 500 companies process applications and run contests. With our new system, we're bringing project management to the next level. Users can create their own business applications using our form builder. Our RESTful API and JavaScript plugin architecture allows developers to build even more powerful apps and integrate WizeHive with other systems.

REST API

WizeHive provides a RESTful API which maps virtually every resource in the platform to a list of URIs. These URIs can be retrieved and manipulated with HTTP verbs.The API is made available for a wide variety of use cases, including commercial applications. Some possibilities include, JavaScript plugins, mobile clients, integration with third party web & mobile applications and more.
